## What the paid bundle is for

Buy only if you need a small, immediately usable starter pack for teams that let Claude Code, Codex, Cursor, OpenHands, MCP, or similar agents touch real repos and want a preflight receipt before work starts.

Skip it if you only need a generic prompt list, if you cannot use local Markdown/checklist assets, or if you expect custom implementation support included in the download.

## Privacy and support expectations

- You do **not** need to send private code, secrets, logs, or credentials to use the bundle.
- The free repo shows the public style and maintenance trail before payment.
- Pre-sale questions should use the public support/trust route linked above so buyers can verify the seller is reachable without private pressure.
